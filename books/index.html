不好，特别是看过前几本坏小孩、长夜难明之后，相对来说，这一本无论是人物塑造还是故事逻辑性上展现的都感觉更糟了，不过也可能是这一本写在其他之前吧。整体看下来，严良对案件关注的动机及其后续反反复复的跳脱让这个人物完全立不起来，只让人感觉生硬厌烦。另外，看完这一本，陈的小说里让我一直感觉少点什么的点越发清晰了，我觉得，那就是对大场面、对外界人情舆论的描写基本没有，这就表现为明明发生或者出现了大事件，警察和严良都全力在办案，但给人的感觉就是没有压迫感，没有紧张感，没有感受到一股故事应该给予读者的一种紧张刺激的探索欲，这就导致，故事真的，没意思。

[美国] J. D. 塞林格 / 施咸荣 / 译林出版社 / 1997-2 / 7.80元
2020-04-08 读过 / 青春 经典 麦田里的守望者 塞林格 / ★★★★☆ 推荐
怀着不平静的心态翻看此书，想要沉浸进书中的世界，很难。颓废的青春、杂乱的言语、幼稚麻木的活着，随心所欲的文字，书写的我所抗拒的语言。越读，只感觉越难以平静，因为，或多会少，霍尔顿的身上，藏着自己的影子，藏着过往的、现在的、未来的自己，为着些许微小的事难过，做着孔乙己般的自欺欺人。但是，读到最后，我也是确实喜欢上了菲比这个小家伙，如果真的有那悬崖前的麦田，我也愿再次守望，体会这快乐。待到下次静下心来在阅读，也许，可以真正的了解吧。2020.3.29

1Q84 BOOK 2
[日] 村上春树 / 施小炜 / 南海出版公司 / 2010-6 / 36.00元
2019-03-16 读过 / 村上春树 1Q84 文学 悬疑 爱情 / ★★★★☆ 推荐
同青豆一样，天吾为这个世界取了它该有的名字“猫城”，一开始这只是他的臆想，但最后他还是发现了两个月亮，原来那些习以为常的毫不起眼的事物，往往能给予你最真实的事情真相；青豆还是动手了，杀死了领袖，尽管原因与初衷有所出入！但结果一样，这是为了所爱的人。我很喜欢她和领袖那场关于世界的话“这个世界上没有绝对的善，也没有绝对的恶。”男人说，“善恶并不是一成不变的东西，而是不断改变所处的场所和立场。”随处可见的话，但他们赋予了它意义“平衡本身就是善”这回归前面“事物归结到底就是善”，我认为，这或许是村上春树想要表达的事物之一，两位主角都直接与人谈论过善恶，世界如何，我也相信如此。
1Q84 BOOK 1
[日] 村上春树 / 施小炜 / 南海出版公司 / 2010-5 / 36.00元
2019-03-16 读过 / 1Q84 村上春树 文学 悬疑 爱情 / ★★★★☆ 推荐
人最初的记忆是什么？也许大都是从懵懵懂懂开始的，但天吾的最初关于那个已丧失了的母亲，对于那历久弥新的记忆的刻画手法，同样让人深刻，村上春树对人的理解与表达真是十分优秀；另外，“一切艺术，一切希求，以及一切行动与探索，都可以看作是以某种善为目标。因此，可以从事物追求的目标出发，来正确地界定善。”——亚里士多德《尼各马可伦理学》这句书中引用的可以说是直接导致故事开始的话，小松的直接理解是“事物归结到底就是善。善就是一切的归结。”此处过多的直白或辩解亦是无用，人性本善是好事，但为自我的追求切实不正当的套上冠冕堂皇的词汇，也不过是自我安慰摆了
