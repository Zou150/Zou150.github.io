无意中看到了这电影，开始是没兴趣的，但是哈里森那憨憨的笑容和吴珊卓那熟悉的东方面孔吸引了我的好奇。听他们的对话与衣着，初以为是个超级英雄故事，但接下来的故事开展完全出乎了我的意料，这是个带有童话色彩的故事，带着些许幽默，却是催人泪下。细节的展现真的让人心酸不已，无论是吴珊卓那饱含热泪的眼神，还是朋友法庭上激动的道出真相与真情相待，没有轰轰烈烈却真实地让人心疼。不得不说，这个故事直的不错，演员们演的也很棒
心花路放
2019-01-07 / 爱情 公路 人生 搞笑 感人 友情 成长 / ★★★★☆ 推荐
2019的又一个惊喜，宁浩这导演有才。初看以为是一部公路喜剧，因情伤上路，会有诸多惊喜，故事、艳遇、碑酒，确实一路上有搞笑有美女，看到大长腿张俪之后，我一度以为这就是所谓的新的恋情吧。但女主的线呢？直到最后的相遇，这个设计真的绝了，很惊喜，却又合情合理，补充了女主部份，让这份爱恋这场上路真正有了份量。真相大白，原来不是所有的失恋之旅都会遇到新的恋情，也许只是为了祭奠，为了告别旧的恋情，仅此而已。
智取威虎山
2018-12-31 / 剧情 历史 革命 经典 剿匪 / ★★★★☆ 推荐
2018最后的小小惊喜，一部14年的电影，从现在看仍然是十分优秀的。剧情跌宕起伏扣人心弦，人物突出，打虎及紧接着的天王盖地虎这一段真的是十分优秀，让人感觉那个时代背景下的土匪就是这个样子的，林更新也是让人蛮惊喜的

无处可逃 / No Escape
2018-10-22 / 感人 社会 剧情 人性 惊悚 / ★★★★☆ 推荐
偶尔刷到的一部电影，情节带着点惊悚味，但节奏真的是掌握的很不错，围绕着一场社会混乱，主角虽不得已卷入其中，但因为紧紧抓紧家人这一信念，即使第一次看到同胞被杀、第一次杀人、伪装逆行叛乱分子、好友死去都不曾崩溃的男主，在最后也展现了人性软弱的一面，虽最后难免光环笼罩，但回过味来已然接近尾声，然后产生一种强烈的冲动想去了解关于该电影的一切，这是一种难得的魅力。
李茶的姑妈
2018-10-06 / 开心麻花 搞笑 男扮女装 喜剧 / ★★★☆☆ 还行
本来看了些影评觉得这部开心麻花的剧可能会失望。但结果来看，基本还是可以的，可能是因为是和一群兄弟去看的，观影体验要求不一样吧。单就搞笑来说，确实是笑点挺足的，听说是由舞台剧改编而来，有些梗虽然挺黄挺暴力，但和一群兄弟一起笑一起闹，那还是挺久远的记忆了。

快把我哥带走
2018-08-20 / 2018 搞笑 青春 漫画改编 校园 成长 / ★★★★☆ 推荐
关于成长与亲情的一部剧，慢节奏、剧情常常一波三折，虽然前期可能不够吸引人、而且每集后面的强行鸡汤让人不适，但慢慢看下去后会发现，这是今年暑假的难得的一部良心剧，关于成长、关于校园、关于家庭，该剧围绕时分十秒两兄妹展开，可以说是一部真正关于青春校园的青少年欢乐搞笑剧。没有乱贴标签，关于青少年的亲情、友情着墨很重，他们之间的成长变化可以说是较符合当下的一些青少年成长过程中会遇到的一些经历与性格，虽然可能会因为是电视剧的缘故将一些东西放大，但基本上，我觉得还是十分成功的。在15、6岁的年纪，我们可以为了一丁点小事吵架，也可以无缘无故的和好，我们可以会因为初见而把某个人当哥们，也可以转眼把兄弟当枪使，我们可以有朦胧的异性荷尔蒙，这就是青春！
